# Ladder
Welcome To Snake And Ladder Game !! (Version 1)

Steps to run this application
------------------------------
1) Compile the source code to the directory i.e. bin
2) Set class path to this bin directory.
3) Run the application using following command line argument : java main.java.Ladder

Steps to run tests
------------------
1) Set class path to bin directory and additionally include jar files hamcrest.jar and junit.jar in the class path.
2) Run tests using following command : java org.junit.runner.JUnitCore test.TestSnakeAndLadder
